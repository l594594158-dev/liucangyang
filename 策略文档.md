## 五、模式B 仓位管理

```
每轮(2s):
  1. LONG持仓 → 均价 vs 当前价 → ±1% → 平仓
  2. SHORT持仓 → 同上
  3. L信号 → LONG仓位? 均价合并 : 开新仓
  4. S信号 → SHORT仓位? 均价合并 : 开新仓
  5. 幂等挂SL/TP单

规则:
  ✅ 多空并存(LONG+SHORT互不阻塞)
  ✅ 每向仅1均价仓位
  ✅ 同向信号只更新均价
  ❌ 无冷却期/无间隔/无移动止盈
```
